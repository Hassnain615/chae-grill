# Chai & Grill Restaurant Management System

## Project Documentation

### Features
- Manage restaurant operations effectively
- User-friendly interfaces for customers and staff
- Menu management with dynamic updates
- Table reservation and management system
- Order management and tracking
- Payment processing integrations

### Installation Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/Hassnain615/chae-grill.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chae-grill
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

### Usage Instructions
- Access the application in your web browser at `http://localhost:3000`
- Create an account or log in to access the restaurant management dashboard.
- Follow the on-screen prompts to manage the restaurant's operations.

### Technology Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Deployment:** Docker, Heroku

### Database Schema
- **Users Table:** Stores information about users such as name, email, and password.
- **Menu Table:** Contains dish details, prices, and availability.
- **Orders Table:** Tracks order information and statuses.
